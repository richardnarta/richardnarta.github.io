    Selamat Datang Di Website Prototype untuk UTS Pemrograman Web

    Website ini dibuat untuk menyelesaikan soal UTS mata kuliah Pemrograman Web IF3028. Website ini dikerjakan oleh Richard Arya Winarta, dengan NIM 121140035 dari kelas RB. Waktu pengerjaan untuk menyelesaikan website ini adalah 12 jam.

    Berikut adalah spesifikasi untuk tampilan website tugasnya :

    • Halaman Utama (Homepage)
      1. Judul halaman yang sesuai
      2. Menu navigasi untuk berpindah ke halaman lain (Form dan Tabel)
      3. Deskripsi singkat tentang website
    • Halaman Formulir
      1. Buatlah form pendaftaran dengan field: Nama, NIM, Program Studi, Email, Password, Jenis Kelamin, Tanggal Lahir, dan Alamat.
      2. Setiap field harus memiliki label yang jelas.
      3. Gunakan atribut required untuk field yang wajib diisi.
      4. Tambahkan tombol submit di bagian bawah form.
      5. Menggunakan method get dan mendesain query string yang akan dikirim
    • Halaman Tabel
      1. Buatlah tabel yang menampilkan daftar mahasiswa dengan kolom: Nama, NIM, Program Studi, Email, Password, Jenis Kelamin, Tanggal Lahir, dan Alamat
      2. Tabel harus memiliki header dan setidaknya 5 baris data (menggunakan data dummy)
      3. Gunakan styling CSS untuk memberikan warna latar belakang yang berbeda pada baris genap (even) dan ganjil (odd)
    • Aturan Desain Layout
      1. Buatlah setiap halaman dengan layout 3 baris (header, body, footer).
      2. Pada baris body terdapat (sidebar, konten utama)
      3. Header harus berisi judul website + logo.
      4. Konten utama berisi beberapa paragraf teks.
      5. Sidebar berisi daftar link ke halaman eksternal (misalnya link ke halaman form atau tabel).
      6. Footer berisi informasi hak cipta dan tahun pembuatan website.
    • Aturan CSS
      1. Setiap halaman file html memiliki 1 file css
      2. Gunakan minimal 5 jenis selektor dalam mendesain halaman
